Message: E1908

maxp: Table version must be either 0x00005000 or 0x00010000

Version 0x00005000 and Version 0x00010000 are the only two versions of the maxp table currently defined. Version 0x00005000 indicates a font file containing CFF data stored in a CFF table; Version 0x00010000 indicates a font file containing TrueType outline data in a glyf table. Note that Microsoft requires that only one type of outline data be used in each font.

For more information, see the maxp table specification.
