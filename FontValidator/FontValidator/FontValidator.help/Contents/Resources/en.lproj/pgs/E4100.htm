Message: E4100

GPOS: The AnchorFormat field is invalid

The Anchor table has the following defined formats:

Format 1 consists of a format identifier and a pair of design unit coordinates that specify the location of the anchor point.

Format 2 specifies a format identifier, a pair of design unit coordinates for the anchor point, and an index to a glyph contour point that is on the outline of a glyph.

Format 3 specifies a format identifier, locates an anchor point, and permits fine adjustments to the coordinate values via Device tables (rather than a contour point).

For more information, see the GPOS table specification.
