Message: E2120

OS/2: sFamilyClass class ID is undefined

Only Family Class ID's 0 through 14 are defined (with several of those reserved).

The font class and font subclass are registered values assigned by IBM to each font family. This parameter is intended for use in selecting an alternate font when the requested font is not available. The font class is the most general and the font subclass is the most specific.

For more information, see the IBM font class guidelines.
