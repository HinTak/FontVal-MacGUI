Message: W2110

OS/2: ySubscriptYOffset is less than zero which is an unlikely value

This value, measured in Funits, should represent the horizontal offset for subscript characters for this font, as recommended by the font designer. The offset is measured from the character origin of the font to the character origin of the subscript character. For upright characters, this value is usually zero. Use a table editing tool to correct this value.

For more information, see the OS/2 table specification.
