Message: E4310

BASE: The BaseTagList table contains an invalid tag

All tag names consist of four characters. Names with less than four characters are allowed if followed by the necessary trailing spaces. All tag names defined within a font (including table names, feature tags, and language tags) must be composed of printing characters represented by ASCII values 32-126.

The Baseline Tags section of the OpenType Tag Registry lists currently registered baseline tags. To learn more about registered tags, see the OpenType Tag Registry

For more information, see the BASE table specification.
