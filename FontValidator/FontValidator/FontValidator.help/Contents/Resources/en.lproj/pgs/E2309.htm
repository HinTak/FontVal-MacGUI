Message: E2309

post: The italicAngle is zero, but the hhea.caretSlopeRun is not zero

The current setting of the caretSlopeRun in the hhea table suggests that the font is italic, but the italicAngle value in the post table indicates an upright font. Ensure that italic designation is consistent within the font by making the italicAngle non-zero, or by setting the caretSlopeRun value in the hhea table to zero.

For more information, see the post table specification.
