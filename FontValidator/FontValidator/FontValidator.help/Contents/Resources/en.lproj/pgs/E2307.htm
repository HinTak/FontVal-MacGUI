Message: E2307

post: The italicAngle does not match the hhea.caretSlopeRise/hhea.caretSlopeRun angle

The angle of the caret should equal the italicAngle of the font so that the insertion point nests well between glyphs. There are cases where these values will not be exactly equal, due to rounding -- but they should be as close as possible. Correct the italicAngle or the caretSlopeRise/Run values to be consistent.

For more information, see the post table specification.
