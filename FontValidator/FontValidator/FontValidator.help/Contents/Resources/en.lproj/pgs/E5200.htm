Message: E5200

OTL Device: The device table's DeltaFormat value is invalid

The DeltaFormat value specifies the format of the DeltaValue array. The DeltaFormat may have one of the following values:

Type Name Description
1 2 Signed 2-bit value, 8 values per uint16
2 4 Signed 4-bit value, 4 values per uint16
3 8 Signed 8-bit value, 2 values per uint16

For more information, see the OTL common table formats.
