Message: E2300

post: An entry in the glyphNameIndex - 258 is greater than the number of names

The glyphNameIndex field should contain a value indicating the ordinal number of the glyph in the post string tables. For values in the range of 258-32767, the number of strings in the post table should equal (glyphNameIndex - 258). Regenerate this font.

For more information, see the post table specification.
