Message: E2124

OS/2: The sxHeight field is zero, but the U+0078 character is mapped to a glyf

If a glyph is encoded in the position of U+0078 (LATIN SMALL LETTER X), then the sxHeight field should be set equal to the top of the unscaled and unhinted glyph bounding box of the glyph (in FUnits).

For more information, see the OS/2 table specification.
