Message: E0700

EBLC: the bitDepth field is not a valid value

The Microsoft rasterizer v.1.7 or greater supports the following bitDepth values: 1 (black/white), 2 (4 levels of gray), 4 (16 levels of gray), and 8 (256 levels of gray). Change this value using a table editing tool or an embedded bitmap editor.

For more information, see the EBLC table specification.
