Message: E2308

post: The italicAngle is nonzero, but the head.macStyle italic bit is not set

Ensure that italic designation is consistent within the font. One case where these designations may differ is in the encoding of a font that has a slope, such as a script face. Such a font will have the italicAngle set, but since it is not considered italic within its own family, the italic bit may not be set.

Use a table editing tool to set the macStyle italic bit (bit 1), or enter zero for the italicAngle in the post table to indicate the font is upright.

For more information, see the post table specification.
