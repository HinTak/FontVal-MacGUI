Message: E2305

post: The isFixedPitch field is zero, but the OS/2 panose bProportion is monospace

Ensure that the designation of fixed-pitch within the font is consistent. Currently, the value for the isFixedPitch in the post table indicates that the font is not monospaced, but the PANOSE bProportion field in the OS/2 table indicates that the font is monospaced. Use a table editing tool to correct values of the isFixedPitch field and the bProportion field so that they agree.

For more information, see the post table specification.
