Message: E4004

GDEF: The CaretValueFormat is invalid

The CaretValueTable can have the following formats:

Format 1 consists of a format identifier, followed by a single coordinate for the caret position.

Format 2 specifies the caret coordinate in terms of a contour point index on a specific glyph.

Format 3 also specifies the value in design units, but uses a Device table rather than a contour point to adjust the value.

For more information, see the GDEF table specification.
