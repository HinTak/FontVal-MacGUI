Message: E1601

kern: The subtable's rangeShift value does not match the calculated value

The entrySelector value, rangeShift value, and searchRange value are used to conduct an efficient binary search of the kerning list. The rangeShift value should equal (the number of kerning pair entries)-(largest power of 2<=(the number of kerning pair entries)*(size of an entry in bytes).

For more information, see the kern table specification.
