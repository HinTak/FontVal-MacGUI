Message: E1800

LTSH: The numGlyphs field does not equal maxp.numGlyphs

Recreate the LTSH table so that the numGlyphs value equals the number of glyphs specified in the maxp table, and that there are yPel values defined for every glyph in the font. The yPel value specifies the ppem for each glyph at which the scaling becomes linear

For more information, see the LTSH table specification.
