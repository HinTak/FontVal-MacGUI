Message: E2113

OS/2: A unicode range was indicated in ulUnicodeRange, but the font has no characters in that range

This error may appear if an incorrect unicode range bit was set, or if there has been a change to the characters within the font after the range bit was set. Use a table-editing tool to change the ulUnicodeRange value to reflect the character coverage of the font.

For more information, see the OS/2 table specification.
