Message: E1304

head: Non-linear scaling flag (bit 4) is clear, but LTSH table is present

Setting bit 4 of the flags field to clear indicates that the advance width of glyphs will scale linearly because hinting instructions will not alter the advance widths. If this is so, then no LTSH table is needed in the font. The presence of the LTSH table indicates that the advance width of glyphs within this font may be affected by hinting instructions. Set this bit to indicate that the advance widths might not scale linearly.

For more information, see the head table specification.
