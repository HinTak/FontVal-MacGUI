Message: W2106

OS/2: The version number is valid, but less than 5

The version number of the OS/2 table identifies the content and layout of the table. The version number used currently is five (5). Versions zero (0, TrueType rev 1.5), one (1, TrueType rev 1.66), two (2, OpenType rev 1.2), three (3, OpenType rev 1.4) and four (4, OpenType rev 1.6) have been used previously. Incrementing the version number of the table will require that values stored in the table be changed to conform to the new version.

For more information, see the OS/2 table specification.
