Message: E1326

head: The xMin value does not match the minimum glyph xMin

The bounding box values should be computed using only glyphs that have contours. Glyphs with no contours should be ignored when identifying max and min values.

This error may indicate that the font is corrupted, or that the head table in the font was not updated after additional glyphs were added and the font should be regenerated.

For more information, see the head table specification.
