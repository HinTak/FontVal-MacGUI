Message: W1500

hmtx: The leftSideBearing is greater than the advance width (unlikely value)

Most glyphs have an advance width that equals the following:

aw = lsb + width of outline + rsb. [the width of the outline being max x coordinate of outline data - min x coordinate of outline data).

However, there are cases where placement of a glyph does not cause the 'penpoint' to be advanced (as with some combining marks, or scripts that require diacritics that are separate from composite glyphs). In these cases, the leftSideBearing may be greater than the advance width.

For more information, see the hmtx table specification.
