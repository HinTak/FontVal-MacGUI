Message: W5021

OTL Device: The device table StartSize is unusually large

The device table defines single pixel adjustments for specified font sizes (in ppem). Since 16384 is the largest unitsPerEm value allowed for OpenType fonts, 16384 is the largest ppem size that can be specified as a starting or ending size in the device table. To learn more about allowable and recommended ppem values, see the head table specification and the Hinting and Production Guidelines Specification.

For more information, see the OTL common table formats.
