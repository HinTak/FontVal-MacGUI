Message: E2306

post: The italicAngle is nonzero, but the hhea.caretSlopeRun is zero

The current setting of the caretSlopeRun in the hhea table suggests that the font is upright, but the italicAngle value in the post table indicates an italic font. Ensure that italic designation is consistent within the font by making the italicAngle zero, or by setting the caretSlopeRun value in the hhea table to non-zero.

For more information, see the post table specification.
