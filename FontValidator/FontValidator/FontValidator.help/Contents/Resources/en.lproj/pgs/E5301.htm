Message: E5301

OTL Feature List: The FeatureRecord tag is invalid

The FeatureTag contained in the FeatureRecord is invalid. All tag names consist of four characters. Names with less than four characters are allowed if followed by the necessary trailing spaces. All tag names defined within a font (including table names, feature tags, and language tags) must be composed of printing characters represented by ASCII values 32-126.

For more information, see the details about OpenType files.
