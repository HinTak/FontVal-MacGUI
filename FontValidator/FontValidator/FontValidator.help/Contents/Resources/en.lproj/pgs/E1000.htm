Message: E1000

gasp: Some rangeGaspBehavior fields contain invalid flags

The four currently defined flags for rangeGaspBehavior include: GASP_DOGRAY (0x0002, use grayscale rendering); GASP_GRIDFIT (0x0001, use gridfitting); GASP_DOGRAY|GASP_GRIDFIT (0x0003, use both grayscale rendering and gridfitting); neither (0x0004). Correct the values using a table editing tool.

For more information, see the gasp table specification.
