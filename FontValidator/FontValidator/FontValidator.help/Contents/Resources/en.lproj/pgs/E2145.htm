Message: E2145

OS/2: The sTypoAscender value is less than or equal to zero

The sTypoAscender value is used to determine the optimum offset from the top of a text frame to the first baseline. The sTypoDescender value should be greater than zero, and the value of (sTypoAscender - sTypoDescender) is recommended to equal one em. To learn more about this value, see the Recommendations for OpenType Fonts.

For more information, see the OS/2 table specification.
