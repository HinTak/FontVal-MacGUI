Message: W1900

maxp: maxSizeOfInstructions computation not via either approved method

The maxSizeOfInstructions field may be calculated by two methods, the first being preferred:

  1. The maximum instruction size of any glyph, taking into account only the instructions in the glyf table.
  2. The same as method #1, plus the size of the prep table plus the size of the fpgm table.

A font showing this warning has maxSizeOfInstructions that is larger than the value calculated by method #1, but not equal to the value calculated by method #2. (Note that a maxSizeOfInstructions value smaller than that calculated via method #1 is an error.)

The fact that the maxSizeOfInstructions value does not match the value computed by either method may indicate some other problem in the font.

For more information, see the maxp table specification.
