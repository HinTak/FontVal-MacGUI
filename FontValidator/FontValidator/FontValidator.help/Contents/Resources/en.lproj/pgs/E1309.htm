Message: E1309

head: The indexToLocFormat value is not 0 or 1

This value should be set to indicate which version, long or short, of the loca table is being used. A value of 0 indicates the loca table uses short offsets, a value of 1 indicates long offsets. Ensure that the tool you are using to edit this font is setting this value correctly.

For more information, see the head table specification.
