Message: E2005

name: The Platform Specific Encoding ID is out of range

For Platform 0 (Unicode) the following Encoding ID's are valid:

Encoding ID Description
0 Unicode 1.0 semantics
1 Unicode 1.1 semantics
2 ISO 10646:1993 semantics
3 Unicode 2.0 and onwards semantics, Unicode BMP only.
4 Unicode 2.0 and onwards semantics, Unicode full repertoire.


For Platform 3 (Microsoft) the following Encoding ID's are valid:

Platform ID Encoding ID Description
3 0 Symbol
3 1 Unicode BMP only
3 2 ShiftJIS
3 3 PRC
3 4 Big5
3 5 Wansung
3 6 Johab
3 7 Reserved
3 8 Reserved
3 9 Reserved
3 10 Unicode full repertoire

For more information, see the name table specification.
