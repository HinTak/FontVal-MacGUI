Message: W1800

LTSH: The LTSH yPel value is zero

The yPel value of the LTSH table records the ppem for each glyph at which the scaling becomes linear, despite instructions effecting the advance width. At and above the recorded threshold size, the glyph must remain linear in its scaling. The proper yPel value for glyphs that scale linearly at all sizes is '1' (at 1 ppem and above the glyph scales linearly). Use a tool such as CacheTT, available from Microsoft, to regenerate the LTSH table with correct values.

For more information, see the LTSH table specification.
