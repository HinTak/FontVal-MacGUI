Message: E2126

OS/2: The font doesn't contain a 3,0 cmap and doesn't contain characters mapped to the 0xf000-0xf0ff range, but the CodePageRange bit #31 (symbol character set) is set

If the symbol bit is set, and the font file contains a 'cmap' subtable for platform of 3 and encoding ID of 1, then all of the characters in the Unicode range 0xF000 - 0xF0FF (inclusive) will be used to enumerate the symbol character set. If the bit is not set, any characters present in that range will not be enumerated as a symbol character set.

For more information, see the OS/2 table specification.
