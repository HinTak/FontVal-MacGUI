Message: E2146

OS/2: The sTypoDescender value is greater than zero

The sTypoDescender value is used to determine the optimum offset from the last baseline to the bottom of the text frame. The sTypoDescender value should be negative, and the value of (sTypoAscender - sTypoDescender) is recommended to equal one em. To learn more about this value, see the Recommendations for OpenType Fonts.

For more information, see the OS/2 table specification.
