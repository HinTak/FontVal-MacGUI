Message: E0020

General: Required table is missing

A minimum of eight tables are required in a valid TrueType font: cmap, head, hhea, hmtx, maxp, name, OS/2, and post.

If TrueType data is present in the font, then the following tables are also required: cvt, fpgm, glyf, loca, and prep. If CFF data is in the font, then both the CFF and VORG tables need to be present.

Microsoft also requires that the DSIG table be present in fonts delivered by vendors to Microsoft.

Microsoft further recommends that the gasp table be present in all fonts.To learn more about these recommendations, see the Hinting and Production Guidelines Specification.

For more information, see the details about OpenType files.
