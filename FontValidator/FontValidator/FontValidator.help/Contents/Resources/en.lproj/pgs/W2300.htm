post: The italicAngle value is unlikely

The italic angle value is measured in degrees, counter-clockwise from the vertical. Zero is the value for upright text. Negative values are used for text that leans to the right (forward for left-to-right writing). This value should be in agreement with the caretSlopeRise and caretSlopeRun values in the hhea table.

For more information, see the post table specification.
