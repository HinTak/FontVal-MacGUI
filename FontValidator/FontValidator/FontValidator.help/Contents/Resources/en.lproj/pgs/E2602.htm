Message: E2602

vhea: The version is not valid

Two versions of the vhea table are currently defined:

Version 1.0 (0x00010000) defines ascent as the distance from the centerline to the previous line's descent; descent as the distance from the centerline to the next line's ascent; and lineGap as reserved (0).

Version 1.1 (0x00011000, note the representation of a non-zero fractional part in Fixed numbers) defines vertTypoAscender, vertTypoDescender, and vertTypoLineGap as functions of the ideographic em-box.

For more information, see the vhea table specification.
