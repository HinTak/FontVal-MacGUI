Message: E2135

OS/2: The xAvgCharWidth field does not equal the calculated value

The xAvgCharWidth value for non-symbol fonts specifies the arithmetic average of the width of all the non-zero width glyphs in a font (change per spec version 1.4). For symbol fonts, the xAvgCharWidth should be calculated as the unweighted average of all mapped glyphs, including those with null contours.

For more information, see the OS/2 table specification.
