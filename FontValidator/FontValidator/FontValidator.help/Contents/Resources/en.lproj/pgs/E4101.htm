Message: E4101

GPOS: The array order is incorrect

Use an OpenType Layout editing tool to create properly structured tables, or regenerate the font if you believe it may have become corrupted. Some of the GPOS arrays and their orderings are listed below:

PairSetOffsets - ordered by Coverage index
PairValueRecords - ordered by GlyphID of the second glyph
Class1Records - ordered by Class1 value
Class2Records - ordered by Class2 value
BaseRecords - ordered by BaseCoverage index
BaseAnchors - ordered by Class
LigatureAttachOffsets - ordered by LigatureCoverage index
ComponentRecords - ordered in writing direction
LigatureAnchors - ordered by Class
Mark2Records - ordered by Coverage
PosRuleSets - ordered by Coverage
PosRules - ordered by preference
PosLookupRecords - ordered by design order
PosClassSets - ordered by Class
PosClassRules - ordered by preference

For more information, see the GPOS table specification.
