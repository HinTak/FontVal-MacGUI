Message: E5002

OTL Class Definition: The RangeRecord array is not in sorted order

Each RangeRecord consists of a start glyph index (Start), an end glyph index (End), and the Coverage Index associated with the range's Start glyph. Ranges must be in starting GlyphID order, and they must be distinct, with no overlapping.

For more information, see the OTL common table formats.
