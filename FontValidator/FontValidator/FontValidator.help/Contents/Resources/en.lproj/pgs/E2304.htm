Message: E2304

post: The isFixedPitch field is zero, but hmtx shows the font as fixed pitch

Ensure that the designation of fixed-pitch within the font is consistent. Currently, the value for the isFixedPitch in the post table indicates that the font is not monospaced, but there is only one pair of advance width/left side bearing values in the hmtx table (indicating the font is fixed pitch). Use a table editing tool to correct value of the isFixedPitch field, or fully populate the hmtx table with metric data.

For more information, see the post table specification.
