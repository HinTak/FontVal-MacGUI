Message: W0302

cmap: The table does not contain any Apple subtables

For Microsoft fonts, at least two subtables are required within the cmap table: Format 4 (Windows) and Format 0 (Macintosh) subtables.

Format 4 is a Windows subtable which uses platform id 3 and encoding id 1 if the font is a text face, and encoding id 0 if the font is a symbol font. This format supports two-byte glyph indices.

Format 0 is a Macintosh subtable and uses platform id 1 and encoding id 0. This format supports single-byte glyph indices.

For more information, see the cmap table specification.
