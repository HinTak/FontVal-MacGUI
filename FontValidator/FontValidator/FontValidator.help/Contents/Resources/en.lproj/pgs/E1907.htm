Message: E1907

maxp: Table version is 1.0 but font does not contain the required glyf table

Version 0x00010000 of the maxp table is used only for fonts containing TrueType outlines stored in a glyf table, not for fonts containing CFF data in a CFF table. Either add a glyf table to the font and keep the version number as is, or change the version number to 0x00005000 to reflect the presence of CFF data in the font. Correct this field using a table editing tool. Note that Microsoft requires that only one outline format be stored in a font.

For more information, see the maxp table specification.
