Message: E0011

offset: The sfnt field is invalid

The sfnt field may have a value of either 1.0 (indicating that TrueType outlines are in the font) or 'OTTO' (indicating CFF data is in the font). The Apple specification for TrueType fonts allows for 'true' and 'typ1' for sfnt version. These version tags should not be used for fonts which contain OpenType tables.

For more information, see the details about OpenType files.
