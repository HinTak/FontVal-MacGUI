Message: W0307

cmap: Characters are mapped in the Unicode Private Use area

Characters are mapped to the Private Use Area. Microsoft recommends that use of the PUA is avoided. For Microsoft fonts, any alternate glyphs, OpenType glyphs, or unused glyphs should NOT be mapped in the Private Use Area. These glyphs should remain unmapped.

For legacy reasons, the 'fi' and 'fl' ligatures are the only glyphs that should be mapped in this area, as follows:

fi ligature
Unicode: U+FB01 and U+F001

fl ligature
Unicode: U+FB02 and U+F002

For more information, see the cmap table specification.
