Message: E4303

BASE: The format of the BaseCoord table is not a valid value

Use a table editing tool to correct this value, or recreate the BASE table using valid formats for the BaseCoord tables. There are currently three formats for the BaseCoord table:

Format 1 consists of a format identifier, followed by a single design unit coordinate that specifies the BaseCoord value. This format has the benefits of small size and simplicity, but the BaseCoord value cannot be hinted for fine adjustments at different sizes or device resolutions.

Format 2 specifies the BaseCoord value in design units, but also supplies a glyph index and a contour point for reference. During font hinting, the contour point on the glyph outline may move.

Format 3 specifies the BaseCoord value in design units, but it uses a Device table rather than a contour point to adjust the value.

For more information, see the BASE table specification.
