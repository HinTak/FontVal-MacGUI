Message: A2100

OS/2: A range in the ulCodePageRange field cannot be validated because a code page is not installed

This message does not indicate there is an error in the font. To validate the codepage compliance of a font, all codepages covered by the font must be installed on the testing system.

To install additional codepages in Windows, do the following:

  1. Go to Control Panel.
  2. Choose Regional and Language Options.
  3. Click the Advanced tab.
  4. In the Codepage Conversion Tables list, select the codepages you wish to install. You will need to have your Windows installation disk available.

For more information, see the OS/2 table specification.
