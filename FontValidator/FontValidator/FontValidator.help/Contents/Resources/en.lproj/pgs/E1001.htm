Message: E1001

gasp: The gaspRange array does not have a 0xFFFF sentinel

The last record should use 0xFFFF as a sentinel value for rangeMaxPPEM and should describe the behavior desired at all sizes larger than the previous record's upper limit. If the only entry in 'gasp' is the 0xFFFF sentinel value, the behavior described will be used for all sizes. Correct the gasp table using a table editing tool that builds the table properly.

For more information, see the gasp table specification.
