Message: W1305

head: The lowestRecPPEM value may be unreasonably small

This value should indicate the smallest size (in pixels) at which the font is readable. The lowestRecPPEM value should be an indicator of the level at which the font is hinted. This value is not required when bitmaps exist in the font for use at small sizes. Use a head table editor to correct this value.

For more information, see the head table specification.
