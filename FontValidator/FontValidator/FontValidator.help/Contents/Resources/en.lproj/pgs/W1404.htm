Message: W1404

hhea: The LineGap value is less than the recommended value

For the most consistent performance between platforms, the linegap value should indicate the difference between (winascent + windescent, lineheight used by Windows) and (hhea.ascender + (-hhea.descender), lineheight used by Apple). This is not an 'interline gap' value.

For more information, see the hhea table specification.
