Message: E1404

hhea: The numberOfHMetrics value is not consistent with the length of the hmtx table

The numberOfHMetrics field must have a value that reflects the number of HMetric entries in the hmtx table. An HMetric entry consists of paired advance width and left side bearing values for a glyph. If the font is monospaced, Microsoft recommends that there be three long-metric entries in the hmtx table, followed by the short metric (lsb) entries. The last long-metric entry in the hmtx table applies to all subsequent glyphs.

Ensure that the tool you are using is updating the value in the hhea table after compressing entries from long-metrics format to short-metrics.

For more information, see the hhea table specification.
