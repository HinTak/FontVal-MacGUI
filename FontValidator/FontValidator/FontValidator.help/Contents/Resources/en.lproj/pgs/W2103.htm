Message: W2103

OS/2: The fsSelection bold bit is set, but is inconsistent with the usWeightClass value

This warning is generated when there is an inconsistency between the usWeightClass field in the OS/2 table and the way the bold bit is set in the fsSelection field of the OS/2 table. If an application is choosing fonts based on weight class values, this font may not be enumerated properly.

If the bold bit is set, the Font Validator expects to find a usWeightClass value of at least 500 (400 being considered 'Regular').

For more information, see the OS/2 table specification.
