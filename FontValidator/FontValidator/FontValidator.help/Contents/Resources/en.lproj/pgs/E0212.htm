Message: E0212

CFF: Names and Strings should be us-ascii

Font names and text strings in the CFF table should be in us-ascii encoding. In particular, copyright signs, trademarks, notices and non-English names of authors should be encoded with the proper encoding in the top level 'name' table. Some font vendors had in the past occasionally and mistakenly put copyright signs in either Latin1 or UTF8 encoding in the CFF String table.

For more information, see the CFF table specification.
