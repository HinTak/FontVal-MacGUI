Message: E1903

maxp: The numGlyphs value doesn't match the number of entries in the loca array

The number of entries in the loca table should be the same as the number of glyphs specified in the maxp table, +1 (In order to compute the length of the last glyph element, there is an extra entry after the last valid index.)

Regenerate the loca table with an outline editing tool.

For more information, see the maxp table specification.
