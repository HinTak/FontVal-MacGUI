Message: E1904

maxp: Table version is 0x00005000 but font contains a glyf table

Version 0x00005000 of the maxp table is used only for fonts containing CFF data stored in the CFF table, not TrueType outlines stored in a glyf table. Either remove the CFF data in the font and change the version to 0x00010000, or remove the glyf table and keep the version number as is. Correct this field using a table editing tool. Note that Microsoft requires that only one outline format be stored in a font.

For more information, see the maxp table specification.
