Message: E0502

DSIG: The digital signature of the file did not verify

The digital signature is not valid, or the file does not match the digital signature. This may indicate that the contents of the font file were altered after the font was signed, or the signature is not current, or a non-matching DSIG table has been appended to the font.

Although the enforcement of signatures is an administrative policy issue, Microsoft recommends that, to promote file integrity, all fonts be digitally signed. Resign the font, or contact the font vendor for a valid version of the font.

For more information, see the DSIG table specification.
