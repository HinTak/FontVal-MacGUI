Message: E0334

cmap: Default UVS table value ranges overlap or out of order

The Unicode Value Ranges of a Default UVS Table must be sorted in increasing order of 'startUnicodeValue'. The ranges must not overlap, i.e., ('startUnicodeValue' + 'additionalCount') must be last than the 'startUnicodeValue' of the following range (if any).

For more information, see the cmap table specification.
