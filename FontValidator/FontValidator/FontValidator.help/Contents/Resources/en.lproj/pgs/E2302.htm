Message: E2302

post: The isFixedPitch field is nonzero, but hmtx shows the font as proportional

Ensure that the designation of fixed-pitch within the font is consistent. Currently, the value for the isFixedPitch in the post table indicates that the font is monospaced, but there are multiple pairs of advance width/left side bearing values in the hmtx table (indicating the font is not fixed pitch). Use a table editing tool to correct value of the isFixedPitch field, or remove all but one pair of metric data from the hmtx table.

For more information, see the post table specification.
