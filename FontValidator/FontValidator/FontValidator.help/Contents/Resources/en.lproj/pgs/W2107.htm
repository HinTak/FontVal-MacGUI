Message: W2107

OS/2: yStrikeoutPosition has unlikely value

This field should specify the position of the top of the strikeout stroke relative to the baseline in font design units. Positive values represent distances above the baseline, while negative values represent distances below the baseline (although this would be an unlikely location for strikeout positioning). A value of zero falls directly on the baseline, while a value of one falls one pel above the baseline. The value of the strikeout position should not interfere with the recognition of standard characters, and therefore not line up with crossbars in a font. For a Roman font with a 2048 em square, 460 is suggested.

For more information, see the OS/2 table specification.
