Message: W1701

loca: Loca references a glyf entry which length is not a multiple of 4

The local offsets to glyph data should be long-aligned, i.e., multiples of 4. Offsets which are not long-aligned may degrade the performance of some processors. Ensure that the outline editing tool you are using is creating the glyf table properly.

For more information, see the loca table specification.
