Message: E1906

maxp: Table version is 1.0 but font contains a CFF table

Version 0x00010000 of the maxp table is used only for fonts containing TrueType outlines stored in a glyf table, not for fonts containing CFF data in a CFF table. Either remove the CFF data in the font and change and leave the version as is, or remove the glyf table and change the version number to 0x00005000. Correct this field using a table editing tool. Note that Microsoft requires that only one outline format be stored in a font.

For more information, see the maxp table specification.
