post: The post name does not match the name in the Adobe Glyph List

PostScript character set glyph names should match those in the PostScript Reference Manual. Glyphs outside the PostScript character set may use a character string in the format of uniXXXX as their PostScript name. To learn more about PostScript names, see Adobe Glyph List.

For more information, see the post table specification.
