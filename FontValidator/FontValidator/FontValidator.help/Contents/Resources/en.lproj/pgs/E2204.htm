Message: E2204

PCLT: The Style Posture bits have a reserved value of 3

Bits 0-1 of the Style field are used to encode Posture, with the values signifying: 0 - upright; 1 - italic/oblique; 2 - alternate italic. These bits should not be set to indicate a value of 3, as this value is reserved.

For more information, see the PCLT table specification.
