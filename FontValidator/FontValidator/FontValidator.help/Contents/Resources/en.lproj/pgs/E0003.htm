Message: E0003

offset: The file was not recognized as a valid OpenType font file

The Validator did not find a valid 4-byte sfnt version label in the offset table. The allowable values are '0100' (font contains TrueType outlines), 'OTTO' (font contains CFF data), or 'ttcf' (TrueType Collection file).

For more information, see the details about OpenType files.
