Message: E2315

post: The version number is invalid

The post table has four possible versions: version 1.0 (TrueType-based font file containing 258 glyphs in the standard Macintosh set); version 2.0 (TrueType-based font file, version 2.5 (deprecated as of OpenType Specification v1.3), and version 3.0, used by OpenType fonts with TrueType or CFF data.

For more information, see the post table specification.
