Message: W2111

OS/2: ySubscriptYSize has unlikely value

This value, measured in Funits, should represent the vertical size for subscripts for this font, as recommended by the font designer. This value can be used by clients for simulating or substituting subscript glyphs if true subscript glyphs are not available in the font. Use a table editing tool to correct this value.

For more information, see the OS/2 table specification.
