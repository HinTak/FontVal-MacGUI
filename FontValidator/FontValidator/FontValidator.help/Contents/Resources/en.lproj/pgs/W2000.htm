Message: W2000

name: The subfamily string is inconsistent with the style of the font

This warning is generated when there is an inconsistency between the contents of the subfamily string in the name table and the way the bold/italic bits are set in the fsSelection field of the OS/2 table. Only bold and italic styling is considered. For example, if the bold bit is set, the subfamily string should read 'Bold'. If the bold bit is not set, the subfamily string should read 'Regular'. To learn more about the fsSelection field in the OS/2 table, see the OS/2 table specification.

For more information, see the name table specification.
