OS/2: Font is a not a symbol font, but PANOSE(tm) byte 1, FamilyType, is set to Latin Symbol

PANOSE(tm) values are important for font clients that must substitute a font when a specified font is not available. PANOSE values should be carefully assigned to ensure adequate font mapping. PANOSE values are fully described in the reference noted below.
