Message: E1100

glyf: Component's transform is ambigously defined

When defining composite components of composite glyphs, bits 3 (WE_HAVE_A_SCALE), 6 (WE_HAVE_AN_X_AND_Y_SCALE), and 7 (WE_HAVE_A_TWO_BY_TWO) should be mutually exclusive. This error indicates that more than one of these bits is set.

For more information, see the glyf table specification.
